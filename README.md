
## Getting Started

```bash
git clone
```

## Run server

```bash
npm run dev

```

## Lancer les migrations

```bash
npx  prisma migrate dev
```


## Créer une nouvelle migration

```bash
npx prisma migrate dev --name nom de la migration
```
